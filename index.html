Como usar o ForestDecision

  • Cadastrar Área - Preencha o formulário com nome, município, estado e hectares desmatados. Clique em "Cadastrar Área".
  • Prioridade Automática - O sistema calcula a prioridade: Alta (≥100ha), Média (50-99ha) ou Baixa (<50ha).
  • Recomendação - Após cadastrar, veja a recomendação personalizada do sistema na seção "Recomendação".
  • Pesquisar - Use o campo de pesquisa para filtrar áreas por município.
  • Visualizar Detalhes - Clique em qualquer linha da tabela para ver informações completas.
  • Editar - Clique no ícone para editar uma área existente.
  • Excluir - Clique no ícone para remover uma área.
  • Relatórios - Baixe relatórios em PDF para análise externa.
  • Gráfico - Visualize os dados de desmatamento agrupados por município.
  • Resumo - Acompanhe estatísticas gerais como total de hectares e municípios afetados.
  • Mapas - Clique nos mapas para ampliar e visualizar detalhes das áreas monitoradas.

Os dados são salvos automaticamente no seu navegador.
